# 团队会议倒计时器用户手册

## 简介

团队会议倒计时器是一个专为高效会议管理设计的桌面应用程序。它允许您创建详细的会议议程，为每个议程项分配时间，并在会议过程中进行精确的时间控制。通过直观的图形界面和强大的功能，帮助您的团队会议更加高效、有序。

## 系统要求

- **操作系统**: Windows 7/8/10/11, macOS, 或 Linux
- **Python版本**: Python 3.6 或更高版本
- **内存**: 至少 512MB RAM
- **存储空间**: 至少 50MB 可用空间

## 安装说明

### 方法一：直接运行源代码
1. 确保已安装 Python 3.6 或更高版本
2. 下载所有项目文件到同一目录：
   - `main.py` (主程序)
   - `timer.py` (计时器核心)
   - `task_manager.py` (任务管理器)
   - `csv_handler.py` (CSV文件处理器)
   - `gui.py` (图形界面)
3. 打开命令行终端，导航到文件所在目录
4. 运行命令：`python main.py`

### 方法二：使用预编译版本（如有提供）
1. 下载对应操作系统的可执行文件
2. 双击运行即可，无需安装Python

## 功能概述

### 主要功能
- ✅ 创建和管理会议议程任务
- ✅ 为每个任务分配精确时间
- ✅ 可视化倒计时显示
- ✅ 暂停和恢复计时

## 使用指南

### 启动应用程序
运行程序后，您将看到主界面，包含以下几个主要区域：
- 计时器显示区（顶部）
- 控制按钮区
- 会议议程管理区
- 文件操作区
- 时间调整区

### 创建会议议程

#### 添加任务
1. 点击"添加任务"按钮
2. 在弹出的对话框中输入：
   - **任务名称**：议程项的名称（必填）
   - **分钟**：任务分配的分钟数
   - **秒**：任务分配的秒数
   - **描述**：任务的详细说明（可选）
3. 点击"添加"确认

#### 编辑任务
1. 在任务列表中选择要编辑的任务
2. 点击"编辑任务"按钮
3. 修改任务信息
4. 点击"更新"保存更改

#### 删除任务
1. 在任务列表中选择要删除的任务
2. 点击"删除任务"按钮
3. 在确认对话框中选择"是"

#### 清空所有任务
- 点击"清空任务"按钮可移除所有议程项

### 使用计时器

#### 开始计时
- 确保已添加至少一个任务
- 点击"开始"按钮启动倒计时
- 计时器将按任务顺序自动进行

#### 暂停计时
- 会议过程中需要暂停时，点击"暂停"按钮
- 计时器将暂停在当前任务

#### 恢复计时
- 点击"开始"按钮（此时按钮功能变为恢复）继续计时

#### 停止计时
- 点击"停止"按钮可完全停止计时器
- 计时器将重置到初始状态

### 调整时间

#### 快速时间调整
在会议进行中，您可以使用时间调整按钮：
- **+1分钟**：为当前任务增加1分钟
- **+5分钟**：为当前任务增加5分钟
- **-1分钟**：为当前任务减少1分钟
- **-5分钟**：为当前任务减少5分钟

#### 精确时间调整
- 暂停计时器
- 编辑当前任务的时长
- 恢复计时器继续会议

### 文件操作

#### 导出议程
1. 点击"导出CSV"按钮
2. 选择保存位置和文件名
3. 点击"保存"
4. 系统将生成包含所有任务信息的CSV文件

#### 导入议程
1. 点击"导入CSV"按钮
2. 选择之前导出的CSV文件
3. 点击"打开"
4. 系统将加载文件中的所有任务

**CSV文件格式说明**：
- 文件必须包含表头：Task Name, Duration (seconds), Description
- Duration列为以秒为单位的时间
- 支持UTF-8编码

## 使用场景示例

### 典型团队会议
1. **会前准备**：
   - 创建议程：["项目回顾(5分钟)", "问题讨论(15分钟)", "行动计划(10分钟)"]
   - 导出议程文件分享给参会人员

2. **会议进行中**：
   - 导入议程文件
   - 开始计时，严格按时间进行
   - 如某个议题需要更多时间，使用"+5分钟"按钮调整

3. **会后跟进**：
   - 保存最终的议程和时间分配
   - 用于下次会议参考

### 头脑风暴会议
- 设置较短的每个发言时间段（如3分钟）
- 使用计时器确保每人发言时间均等
- 灵活调整时间分配

## 提示与技巧

### 高效使用建议
1. **提前准备**：在会议开始前创建完整的议程
2. **时间缓冲**：在议程末尾预留5-10分钟缓冲时间
3. **灵活调整**：根据会议进展动态调整时间分配
4. **文件备份**：定期导出议程文件作为备份

### 故障排除

#### 常见问题
**Q: 程序无法启动**
A: 检查Python版本是否为3.6或更高，确保所有必要文件在同一目录。

**Q: 导入CSV文件失败**
A: 检查CSV文件格式是否正确，确保包含必要的列标题。

**Q: 计时器不准确**
A: 这是一个基于软件的计时器，可能受系统负载影响。对于精确计时需求，建议使用专门的硬件计时器。

**Q: 任务列表显示异常**
A: 尝试清空任务并重新添加，或重启应用程序。
